# Vegan_WebProject
> Encore PlayData Web Project  
> 국내 채식 인구 증가로 커져가는 수요에 맞는 편리한 서비스 제공 목적  
> 비건을 위한 웹서비스 제작

## 사용 기술
> BACK
  - Java 1.8 : Servlet Controller 구현
  - Tomcat : 서버연결
  - MySQL : 가게 데이터베이스와 연동 로그인 및 게시판 글
> FRONT
  - Bootstrap : 전체적인 디자인 틀 수립
  - JAVASCRIPT (jQuery, JSON, Ajax)
  - CSS : 각각의 홈페이지 구성에 맞게 디자인
  - HTML : 기본 틀 작성
